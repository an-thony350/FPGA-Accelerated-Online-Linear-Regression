# Source Availability

The source code is private.

## Why the Source Is Not Public

The original archive contains sensitive or non-public material:

- private key material,
- bitstreams and hardware handoff files,
- PYNQ notebooks,
- trading/API code,
- controller/audio code,
- FPGA implementation source,
- coursework/project material.

Publishing those files would be unsafe and inappropriate.

## Public in This Repository

- Accelerator architecture.
- Numeric-format and bus-design notes.
- PYNQ integration summary.
- Testing and validation summary.
- Performance/resource figures.

## Not Published

- RTL/HLS/source files.
- Bitstreams/HWH/TCL files.
- Python trading code.
- Credentials.
- Jupyter notebooks.
- Raw market data or account state.
- Deployable trading automation.

I can discuss the RTL datapath, accumulator sizing, DSP tradeoffs, sample packing, PYNQ integration, and hardware/software validation strategy. Where appropriate, I can provide supervised evidence without publishing sensitive material.
